Module braid::tests [−] [src]

Unit tests for datastore implementations.

These are exported so that datastore implementations outside of the braid crate can reuse them. Generally you can use the convenience macros test_account_impl, test_metadata_impl, and test_transaction_impl rather than referencing these functions directly - but they may be useful to reference directly if you want to implement a datastore that doesn't support full functionality.
