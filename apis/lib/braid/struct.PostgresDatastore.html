Creates a new postgres-backed datastore.

Arguments

  • pool_size - The maximum number of connections to maintain to postgres. If None, it defaults to twice the number of CPUs.
  • connetion_string - The postgres database connection string.
  • secret - A secret value. This is used as a pepper in hashing sensitive account data.
  • secure_uuids - If true, UUIDv4 will be used, which will result in difficult to guess UUIDs at the detriment of a more index-optimized (and thus faster) variant.
